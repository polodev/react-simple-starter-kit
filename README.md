# react-simple-starter-kit
Basic Starter kit for react es6 less

### Usage

```
npm install
npm start
open http://localhost:3000
```

### Dependencies
* react
* react-dom
* react-router

### devDependencies

* babel-loader
* babel-preset-es2015
* babel-preset-react
